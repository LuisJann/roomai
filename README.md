# RoomAI

**RoomAI** is the ultimate AI-powered interior design application. Forget complex software—take a photo, build a millimeter-precise 3D environment, and let Artificial Intelligence furnish the space for you.

[🇮🇹 Leggi in Italiano](#roomai---italiano)

## 🌟 Key Features
- **📱 Mobile Connection:** Scan a QR code with your phone to instantly upload room photos directly to your desktop workspace.
- **📐 Spatial 3D Editor:** Build out your exact floor plan. Place doors, windows, and 3D assets in a precise digital environment to design exact spaces.
- **✨ Artificial Intelligence:** Use generative AI to create dozens of hyper-realistic renders starting from an empty or already furnished room.

## 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org) (React 19)
- **Styling:** Tailwind CSS, Framer Motion for animations
- **3D Rendering:** Three.js, React Three Fiber, React Three Drei
- **Backend & Auth:** Supabase, Vercel KV
- **AI Integrations:** Google GenAI, Hugging Face Inference

## 🚀 Getting Started

First, install the dependencies and run the development server:
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# RoomAI - Italiano

**RoomAI** è l'applicazione definitiva per l'interior design basata sull'intelligenza artificiale. Dimentica i vecchi software complessi: scatta una foto, costruisci l'ambiente in 3D con precisione millimetrica e lascia che l'Intelligenza Artificiale arredi lo spazio per te.

## 🌟 Funzionalità Principali
- **📱 Connessione Mobile:** Scansiona un QR code col tuo telefono e carica istantaneamente le foto dell'ambiente direttamente sul computer.
- **📐 Editor 3D Spaziale:** Posiziona porte, finestre e mobili 3D in un ambiente digitale millimetrico per progettare gli spazi esatti.
- **✨ Intelligenza Artificiale:** Usa l'AI generativa per creare decine di render iper-realistici partendo dalla tua stanza vuota o arredata.

## 🛠️ Tecnologie Utilizzate
- **Framework:** [Next.js](https://nextjs.org) (React 19)
- **Stili:** Tailwind CSS, Framer Motion per le animazioni
- **Motore 3D:** Three.js, React Three Fiber, React Three Drei
- **Backend & Autenticazione:** Supabase, Vercel KV
- **Integrazioni AI:** Google GenAI, Hugging Face Inference

## 🚀 Per Iniziare

Installa le dipendenze e avvia il server di sviluppo:
```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) nel browser per visualizzare l'app.
